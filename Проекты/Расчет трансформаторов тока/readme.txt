Программа для расчета и выбора трансформаторов тока по условиям 10% погрешности, напряжению на вторичной обмотке и времени до насыщения, а также для оформления расчетов.
В основе лежит методика, изложенная в ГОСТ Р 58669-2019.
Один из первых проектов, поэтому многие вещи реализованы не лучшим образом, а комментарии практически отсутствуют. В разработке находится новая версия, но там реализован пока только интерфейс (Jimmie_4).
