Программа для перевода расчетов из Mathcad в Excel. Выполнена на основе готового проекта парсера Mathcad в LaTex:
https://github.com/ArtificialTruth/PTC-Mathcad-to-LaTeX-parser/tree/master
Мною был несколько расширен функционал и добавлена функция подстановки чисел в выражения. Прикладываю пример расчетов, для которых программа задумана
